判断二叉树是否为完全二叉树

若设二叉树的深度为h，除第 h 层外，其它各层 (1～h-1) 的结点数都达到最大个数，第 h 层所有的结点都连续集中在最左边，这就是完全二叉树。

【分析】

1.按层遍历二叉树，从每层的左边向右边依次遍历所有的节点。

2.如果当前节点有右孩子，但没有左孩子，直接返回false

3.如果当前节点并不是左右孩子都有，那之后的节点必须都为叶节点，否则返回false

4.遍历过程中如果不返回false,遍历结束后返回true

【AC代码】

`//判断一棵树是否为完全二叉树
public class Solution {
    public boolean isCBT(TreeNode head){
        if(head==null){
            return true;
        }
        Queue<TreeNode> queue=new LinkedList<TreeNode>();
        boolean leaf=false;
        TreeNode l=null;
        TreeNode r=null;
        queue.offer(head);
        while(!queue.isEmpty()){
            head=queue.poll();
            l=head.left;
            r=head.right;
            if(leaf&&(l!=null||r!=null)||(l==null&&r!=null)){//如果之前某个节点是叶子节点了，之后的节点左右不全为空，则返回false
                return false;
            }
            if(l!=null){
                queue.offer(l);
            }
            if(r!=null){
                queue.offer(r);
            }else{
                leaf=true;
            }
        }
        return true;
    }
    public static void main(String[] args) {
        Solution d=new Solution();
        TreeNode head1=new TreeNode(1);
        TreeNode head2=new TreeNode(2);
        TreeNode head3=new TreeNode(3);
        TreeNode head4=new TreeNode(4);
        TreeNode head5=new TreeNode(5);
        head1.left=head2;
        head1.right=head3;
        head2.left=head4;
        head3.right=head5;
        System.out.println(d.isCBT(head1)?"该二叉树是完全二叉树":"该二叉树不是完全二叉树");

    }
}`
