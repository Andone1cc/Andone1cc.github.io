统计完全二叉树的节点数

2016-11-15

遍历整棵树当然可以求出节点数，但这肯定不是最优解法。

如果完全二叉树的层数为h,时间复杂度为O（h的平方）

【分析】

1.如果head==null,说明是空树，直接返回0。

2.如果不是空树，就求树的高度，求法是找到树的最左节点看能到哪一层，层数记为h

3.递归过程bs(node,l,h).node表示当前节点，l表示node所在的层数，h表示整棵树的层数始终不变。

第一种情况：

找到node右子树的最左节点，如果能到达最后一层，说明node的整棵左子树是满二叉树。其节点数为2的h次方-1.（算上node节点自己）然后再求右子树的节点数bs(node.right,l+1,h)

第二种情况：

找到node右子树的最左节点，没有到达最后一层，说明node的整棵右子树都是满二叉树，其节点数为2的h次方-l-1,然后再求左子树的节点数bs（node.left,l+1,h）

【AC代码】

`//统计完全二叉树的节点数
public class Solution {
    public int nodeNum(TreeNode head){
        if(head==null){
            return 0;
        }
        return bs(head,1,mostLeftLevel(head,1));
    }
    private int bs(TreeNode head, int l, int h) {
        if(l==h){
            return 1;
        }
        if(mostLeftLevel(head.right,l+1 )==h){
            return (1<<(h-l))+bs(head.right,l+1,h);
        }else{
            return (1<<(h-l-1))+bs(head.left,l+1,h);
        }
    }
    private int mostLeftLevel(TreeNode node, int level) {
        while(node!=null){
            level++;
            node=node.left;
        }
        return level-1;
    }
    public static void main(String[] args) {
        Solution d=new Solution();
        TreeNode head1=new TreeNode(1);
        TreeNode head2=new TreeNode(2);
        TreeNode head3=new TreeNode(3);
        TreeNode head4=new TreeNode(4);
        TreeNode head5=new TreeNode(5);
        TreeNode head6=new TreeNode(6);
        TreeNode head7=new TreeNode(7);
        TreeNode head8=new TreeNode(8);
        head1.left=head2;
        head1.right=head3;
        head2.left=head4;
        head2.right=head5;
        head3.left=head6;
        head3.right=head7;
        head4.left=head8;
        System.out.println(d.nodeNum(head1));
    }
}`
