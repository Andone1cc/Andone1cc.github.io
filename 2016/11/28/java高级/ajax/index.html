什么是Ajax?

Ajax全称为”Asynchronous JavaScript and XML”（异步JavaScript和XML）,是一种创建交互式网页应用的网页开发技术。

Ajax的本质是一个浏览器端的技术。

为什么要学习Ajax?

1.通过异步模式，提升了用户体验。

2.优化了浏览器和服务器之间的传输，减少不必要的数据往返，减少了带宽占用。

ajax局部刷新技术，其实就是使用js实现局部数据的实时变更，解决了不同响应之间的数据共享问题。

创建ajax程序的基本流程

1.创建XMLHttpRequest对象(有浏览器差异问题)

var request;
if(window.XMLHttpRequest){//Mozilla
    request=new XMLHttpRequest();
}else if(window.ActiveXObject){    //IE
    request=new ActiveXObject("Msxml2.XMLHTTP");
}

2.使用XMLHttpRequest对象创建请求

request.open("get","testAjaxServlet");

3.复写onreadystatechange函数,监视response的状态,处理服务器返回的数据。

接收响应结果：

响应数据的类型：

1.普通文本：就是字符串

2.json格式的数据：就是使用了json数据格式拼接的字符串，这样在客户端我们就可以使用eval()方法将响应的字符串数据转换成可执行的js代码。极力推荐使用json数据格式来做响应

3.XML数据：使用JSP页面来书写XML数据，设置响应头为XML类型的数据。在客户端使用responseXML来接收响应的xml数据（返回document对象）。使用document对象来操作xml数据。
doc.getElementsByTagName(“标签名”)[角标].firstChild.data

按照readyState的值为4来接收，4表示响应数据成功接收

status：200表示一切OK

404表示资源未找到

500表示服务器内部错误

`request.onreadystatechange=function(){
    //readyState的值为4表示数据成功接收
    if(request.readyState==4){
        if(request.status==200){
            var result=request.responseText;//接收数据
            eval("var arr="+result);//使用eval强转
            /*使用js操作文档结构..*/
        }
}`

4.使用XMLHttpRequest对象创建请求

get：
    请求数据以问号的形式拼接在url后面，键值对。
    因为浏览器差异的问题，在send的时候一定是：send(null)

    request.open("get","uri?用户请求数据");
    request.send(null);        
post：
    post是有请求实体的，用户数据使用send直接发送

    request.open("post","uri");
    request.setRequestHeader("Content-Type","application/x-www-form-urlencoded");
    request.send("用户数据");

ajax应用实例

需求：在一个页面加载完成后，在页面上获取到权限信息，显示到下拉框中。

1.jQuery的ajax

JSP:

`<script type="text/javascript">
    //页面加载的时候发送请求
    $(document).ready(function(){
        getPower();
    })
    //获取权限信息
    function getPower(){

        $.get("user.action",{oper:"power"},function(result){
            //使用eval强转
            eval("var power="+result);
            //获取权限下拉框对象
            var sel=$("#sel");
            sel.empty();
            //遍历
            for(var i=0;i<power.length;i++){
                sel.append("<option value="+power[i].pid+">"+power[i].pname+"</option>");                
            }
        });
    }
</script>    
`

window.onload(function(){

    }
    //2创建请求
